Objetivo del sistema

Desarrollar un sistema interactivo en Python que permita jugar Piedra, Papel o Tijera en dos modalidades: entre dos jugadores o contra el servidor. 
El proyecto tiene como finalidad aplicar los conocimientos adquiridos durante la asignatura, utilizando estructuras de programación para resolver 
un problema mediante una computadora.

Descripción de funcionalidades 

El sistema desarrollado posee las siguientes funcionalidades:
* Menú principal para seleccionar el modo de juego.
* Modalidad de juego entre dos jugadores.
* Modalidad de juego contra el servidor.
* Validación de opciones ingresadas por los usuarios.
* Selección aleatoria de jugadas por parte del servidor.
* Determinación automática del ganador de cada ronda.
* Registro y visualización del marcador.
* Repetición de partidas mediante ciclos.
* Finalización controlada del programa

Introducción

La programación permite diseñar soluciones informáticas para resolver problemas de manera eficiente. En este proyecto se desarrolló el juego Piedra, Papel o Tijera
utilizando el lenguaje de programación Python. Durante su construcción se aplicaron conceptos fundamentales como variables, operadores,
estructuras de decisión, bucles, listas, tuplas, diccionarios y funciones. El sistema ofrece una experiencia interactiva donde los usuarios pueden competir
entre sí o contra la computadora. 

Descripción del problema

El problema planteado consiste en automatizar el juego tradicional Piedra, Papel o Tijera mediante un programa informático. Se requiere que el sistema permita
registrar las elecciones de los jugadores, validar las opciones ingresadas, determinar automáticamente el ganador según las reglas establecidas y mostrar los 
resultados de cada partida. Además, debe permitir la repetición de rondas y llevar un control básico de puntuaciones.

Explicación del sistema desarrollado 

El sistema inicia mostrando un menú principal donde el usuario puede seleccionar si desea jugar contra otro jugador o contra el servidor.
Las opciones disponibles del juego se almacenan en una lista. Las reglas que determinan quién gana se almacenan en un diccionario. Además, 
se utiliza una función llamada “determinar_ganador”, encargada de comparar las elecciones realizadas y devolver el resultado correspondiente.
Cuando se juega contra el servidor, la computadora selecciona una opción aleatoria mediante la librería random. En ambos modos de juego se utilizan
estructuras condicionales para determinar el ganador y ciclos para permitir jugar varias rondas consecutivas.
